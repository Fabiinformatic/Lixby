Política de Devoluciones

Última actualización: Marzo 2026

1. Derecho de desistimiento

De acuerdo con la normativa europea de comercio electrónico, tienes derecho a desistir de tu compra dentro de 14 días naturales desde la entrega del producto sin necesidad de justificación.

Lixby amplía voluntariamente este periodo hasta 30 días naturales desde la entrega del producto para facilitar las devoluciones.

Nota importante: Este derecho no se aplica a productos personalizados, consumibles o servicios digitales descargados.

2. Condiciones para devoluciones

Productos elegibles para devolución:

  • LixBuds One y accesorios
  • Otros productos sin usar o con embalaje original intacto

Condiciones que debe cumplir el producto:

  • Estar en perfecto estado
  • Incluir todos los accesorios originales
  • Acompañarse del comprobante de compra
  • Embalaje original en buen estado cuando sea posible

3. Proceso de devolución

Paso 1: Notificación

Notifica tu intención de devolver el producto dentro del plazo establecido enviando un email a returns@lixby.com con:

  • Número de pedido
  • Motivo de la devolución
  • Datos de contacto

Paso 2: Autorización

Recibirás una autorización de devolución (RMA) junto con las instrucciones de envío.

Paso 3: Envío

Empaca el producto de forma segura y envíalo a:

Lixby S.L. - Departamento de Devoluciones
Calle Principal 123
28001 Madrid, España

Paso 4: Reembolso

Una vez recibido y verificado el producto, procesaremos el reembolso dentro de 14 días hábiles.

4. Costos de envío

Envío inicial gratis: Ofrecemos envío gratuito en compras superiores a 30 €.

  • Producto defectuoso → Lixby paga el envío
  • Cambio de opinión → El cliente paga el envío

5. Productos no reembolsables

  • Productos personalizados
  • Productos con signos evidentes de uso
  • Productos dañados por negligencia del cliente
  • Servicios digitales descargados
  • Suscripciones o membresías activadas

6. Cambios de producto

Si deseas cambiar un producto por otro modelo o color, primero debes procesar la devolución y realizar una nueva compra. También puedes contactar con nuestro soporte para asistencia.

7. Reembolsos parciales

Podrán aplicarse reembolsos parciales si:

  • El producto presenta uso indebido
  • Existen daños causados por el cliente
  • Se requieren costos de restauración

8. Productos defectuosos o dañados

Si recibes un producto defectuoso:

  1. Reporta el problema dentro de 7 días desde la entrega
  2. Adjunta fotografías del problema
  3. Lixby cubrirá el costo del envío
  4. Puedes elegir entre reemplazo o reembolso

9. Garantía

Nuestros productos incluyen la garantía legal de 2 años para defectos de fabricación.

Para reclamaciones contacta con: support@lixby.com
